# Учёт ФОП: сайт + Google Таблица

## Что внутри

- `index.html` — сайт для GitHub Pages: загрузка скринов, OCR, карточки продаж, архив, CSV/PDF, синхронизация с Google Sheets.
- `Code.gs` — backend для Google Apps Script, который пишет данные в твою таблицу.

## Как поставить

### 1. Google Apps Script

1. Открой свою Google Таблицу.
2. Расширения → Apps Script.
3. Удали всё в редакторе.
4. Вставь содержимое `Code.gs`.
5. Сохрани проект.
6. Нажми Deploy / Развернуть → New deployment / Новое развертывание.
7. Тип: Web app.
8. Execute as: Me / От моего имени.
9. Who has access: Anyone / Все.
10. Нажми Deploy и скопируй Web App URL, который заканчивается на `/exec`.

### 2. Сайт

1. В GitHub замени свой `index.html` на файл `index.html` из этой папки.
2. Открой сайт.
3. Перейди в Настройки.
4. Вставь Apps Script Web App URL.
5. Нажми “Проверить связь”.

## Как пользоваться

1. Вкладка “Разбор скринов”.
2. Загрузи все скрины одной кучей.
3. Нажми OCR, если хочешь автоизвлечение текста.
4. Создай карточки продаж или нажми “Собрать по суммам”.
5. Перетащи скрины в карточки.
6. Проверь поля.
7. Нажми “Сохранить готовые”.
8. Записи уйдут в архив и синхронизируются в Google Таблицу.

## Важно

Фото пока хранятся локально в браузере, а в Google Таблицу отправляются ссылки вида `local:id`. Это сделано, чтобы первая версия была простой и стабильной без Google Drive API. Следующий апгрейд — загрузка скринов в Google Drive и сохранение реальных ссылок в таблице.
